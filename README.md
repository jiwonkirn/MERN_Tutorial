# MERN Tutorial

## 개요

MERN(MongoDB, Express, React, Nodejs)를 이용하여 간단한 소셜 네트워크 서비스를 만드는 프로젝트입니다.

## 주요 기능

- 회원가입
- 로그인
- 개인정보 수정
- 댓글 기능
- 이 외의 CRUD 추가 예정

## 사용기술

- Client

  - `Reactjs`
  - `Redux`
  - `Html5`
  - `Css(sass)`

- DATABASE

  - `mongodb` (`mongoose`, `mLab`)

- Server

  - `Nodejs`
  - `Express`

- etc dependencies

  `bcryptjs`, `body-parser`, `express`, `jsonwebtoken`, `mongoose`, `passport`, `passport-jwt`, `validator`, `devDependencies`, `nodemon`
